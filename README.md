# FixVehicle

### What is this?

* This is a resource which you can use to repair your vehicle.

### Features
- Use command **`/fix`** to repair your vehicle.

## Download & Installation

### Using Git
```
cd resources
git clone https://github.com/helmimarif/FixVehicle
```

### Manually
- Download from this.

## Installation
- Add this to your `server.cfg`:

```
start FixVehicle
```
